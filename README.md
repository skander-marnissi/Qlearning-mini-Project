# Qlearning-mini-Project

Small script done with numpy on python 3.8 that uses reinforcement learning so that an agent is capable to cross the grid with the maximum of rewards.

## Installation

Open cmd and type the following commands: 

```bash
git clone https://github.com/SkanderMarnissi/Qlearning-mini-Project/
```

After downloading:

```bash
cd Qlearning-mini-Project
```

## Usage

In order to process the program and make it work you need to:

### Run the script, open your terminal and type:

```bash
python Qlearning_script.py
```
